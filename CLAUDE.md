# CLAUDE.md

## Project Overview

UIGen is an AI-powered React component generator built with Next.js.
It generates UI components using Claude AI and Tailwind CSS.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Prisma + SQLite
- Anthropic API

## Coding Guidelines

- Use functional React components
- Prefer Tailwind for styling
- Keep components reusable and clean
- Use TypeScript where applicable

## Project Structure

- src/app → pages and routes
- src/components → reusable UI components
- src/lib → utilities and helpers
- src/actions → server actions

## Instructions for Claude

- Always generate clean and readable React code
- Use Tailwind CSS for styling
- Avoid unnecessary complexity
- Follow project structure strictly
